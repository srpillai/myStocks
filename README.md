# README

This is Stock Market app that let's you look up stock quotes and financial information and save it to a database.

Uses Ruby on Rails with a database.


## Built With:


* Ruby version 2.6.3
* Rails version 6.0

* CSS & Bootstrap- for styling
* Database sqlite3 version 1.3
